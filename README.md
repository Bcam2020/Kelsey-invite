# 🥩 BeefSmoke House Invite

A single-page dinner invitation site.

## Setup

Just open `index.html` in a browser — no build step needed.

## GitHub Pages (so Kelsey can access it via a link)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. GitHub will give you a URL like `https://yourusername.github.io/repo-name`

Send her that link. Done.
